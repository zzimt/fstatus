Simple status bar.

## Dependencies

- `gcc`
- `make`
- `pkg-config`
- `X11`

## Usage

### Building

```bash
make
```

### Installing

```bash
sudo make install
```

### Configuration

`fstatus` is configured by changing `src/config.h` and rebuilding the application.
